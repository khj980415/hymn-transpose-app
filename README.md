# 🎵 찬송가 전조 도우미

악보 이미지를 MusicXML로 변환하고 전조/계이름을 표시하는 데스크톱 앱

## 🚀 기능

- 📷 악보 이미지 업로드
- 🎼 MusicXML 변환 (OMR)
- 🔄 자동 전조
- 🎶 계이름 표시

## 💻 개발 환경 설정

### 필수 요구사항

- Node.js 18+ 
- npm or yarn

### 설치
```bash
# 의존성 설치
npm install

# 개발 모드 실행
npm run electron:dev

# 프로덕션 빌드
npm run electron:build
```

## 🛠️ 기술 스택

- Vue 3 + Vite
- Electron
- OpenSheetMusicDisplay
- Tonal.js
- Audiveris (OMR)
